Wakati API

Calculate Read Time

Usage

This API calculate the reading time of a sentence. It supports both GET and POST requests. The default reading speed is 238 words per minute. 

curl -X GET "https://wakati.ritoho8508.workers.dev/api/calculate?sentence=This+is+a+sentence+to+calculate+the+reading+time+of."

curl -X POST "https://wakati.ritoho8508.workers.dev/api/calculate" \
-H "Content-Type: application/json" \
-d '{"sentence": "This is a sentence to calculate the reading time of."}'

API Documentation
